# 人脈手札 — Fortune-Based CRM

結合東西方命理分析的人脈管理工具，快速判斷合作適配度。

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `人脈手札-v3.html` | **最新版（推薦使用）** — 完整命理引擎、東西方雙主題、十神/相位系統 |
| `人脈手札-v2.html` | 純東方主題版，深色風格 |
| `人脈手札-v1.html` | 初始桌面版 |

## 使用方式

直接在瀏覽器開啟 `人脈手札-v3.html` 即可使用，無需安裝。

## V3 功能

- **東方命理**：八字（十神、身強弱、天干合化、地支六合六沖）+ 數字易經
- **西方占星**：星座相位（三合/六分/對沖/刑相）+ 守護星 + 元素互動
- **合作分析**：十神關係、場景適配度、互補指標、雙向視角
- **名片 OCR**：拍照自動辨識（需 Claude API Key）
- **語音筆記**：語音輸入備忘
- **資料管理**：IndexedDB 本地儲存 + JSON 匯出匯入

## 技術

單檔 HTML 應用，無外部依賴。資料儲存在瀏覽器 IndexedDB。
