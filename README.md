## 使用方式

### 一般使用者
直接在瀏覽器開啟 `index.html` 或線上網址即可使用，無需安裝。

### 商業分析報告
`商業策略/` 內三個 `.html` 檔可直接在瀏覽器開啟，支援互動式參數調整（拖滑桿即時看試算結果）。

## V3 核心功能

- **東方命理**：八字（十神、身強弱、天干合化、地支六合六沖）+ 數字易經
- **西方占星**：星座相位（三合/六分/對沖/刑相）+ 守護星 + 元素互動
- **合作分析**：十神關係、場景適配度、互補指標、雙向視角
- **名片 OCR**：拍照自動辨識（需 Claude API Key）
- **語音筆記**：語音輸入備忘
- **資料管理**：IndexedDB 本地儲存 + JSON 匯出匯入

## 商業模式摘要（詳見 `商業策略/`）

- **推薦定價**：免費 20 人 → Pro $149 TWD/月（年付 $1,188）
- **損益平衡**：約 107 位付費用戶（~2,100 總註冊）
- **達標時程**：約第 5–8 個月
- **行銷預算**：$30K TWD/月，CAC ~$38/人
- **關鍵增長槓桿**：裂變推薦（零成本）+ SEO 內容（長期資產）+ 命理 KOL 合作

## 技術架構

- 單檔 HTML 應用，**無外部依賴**（Google Fonts 為唯一例外）
- 資料儲存在瀏覽器 **IndexedDB**，不發到伺服器
- 部署：Cloudflare Workers + 靜態資源
