## Stage Gates（階段關卡）

每個階段結束時，**必須通過下面條件才能進下一階段**。沒過就退回打磨，不要硬推。

### 階段 1 → 階段 2（W8 末）
- [ ] 5 個非親友使用 2 次以上
- [ ] 5 條具體文字回饋（不只是「不錯」）
- [ ] App 沒有 P0 bug

### 階段 2 → 階段 3（W16 末）
- [ ] 100 註冊（從 Analytics 不重複 UV 計算）
- [ ] 5 筆贊助（驗證「願意付錢」）
- [ ] 30 天留存 ≥ 10%
- [ ] KOL 合作貼文觸及 ≥ 1000

### 階段 3 → 階段 4（W24 末）
- [ ] Lifetime 賣 ≥ 50 名
- [ ] 月活 ≥ 500
- [ ] 7 日留存 ≥ 25%
- [ ] 每月時間投入 ≤ 20 小時

### 階段 4 → 階段 5（W36 末）
- [ ] 總付費（Lifetime 累計 + Cloud Sync 訂閱）≥ 107
- [ ] Cloud Sync Churn < 8%/月
- [ ] NPS ≥ 30

### 階段 5（W48 末，年度檢核）
- [ ] MRR ≥ NT$29,800
- [ ] 累計付費 ≥ 200
- [ ] 您每月時間投入 < 25 小時，仍有正現金流

## 每週固定動作（不在上表，是底線習慣）

不論哪一週，這些都要做：

- **每天 10 分鐘**：回 Email / Line、看 Analytics 一眼
- **每週日晚 30 分鐘**：填本週進度 + 看下週清單
- **每週五 1 小時**：寫一篇短內容（Threads / IG / 小紅書）

## 怎麼用這份清單

### 推薦工作流

1. **每週日晚上**：打開這份檔案、找到當週、把任務複製到 Notion / Todoist / Apple 提醒事項
2. **週一到週四晚上**：每天花 30 分鐘做一項
3. **週六**：補做沒做完的、寫週末短內容
4. **週日晚上**：打勾完成的、看下週

### 卡關時

任何一週做不完是正常的。優先級：
1. **🛠 Dev 任務** > 📣 Mkt 任務（因為產品不好的話行銷都白做）
2. **本週的 Stage Gate 相關任務** > 其他（會卡住階段推進的優先）
3. **延後一週 ≤ 兩次**（同一任務不要拖超過 2 週，要嘛快做要嘛刪掉）

### 跟 Claude 開工

每週初告訴 Claude：「這週要做 W3 的 1, 2, 4 三項，從 #1 開始。」我會：
1. 把任務拆成可執行步驟
2. 邊改 App 邊跟您確認
3. 完成後幫您寫 commit message + 推上 GitHub
