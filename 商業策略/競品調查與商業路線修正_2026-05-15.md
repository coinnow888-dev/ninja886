## 為什麼是這兩款

您自己說過，人脈手札的 idea 來自：
- **名片王**（人脈/名片管理那一半）
- **先知命局**（命理分析那一半）

所以這兩款不是「對手」，是您產品的兩個**血親**。看它們的價格 = 看您兩個來源市場願意付多少錢。

## 一、名片王（蒙恬科技 PenPower）— 收費機制

### 基本資料
- **公司**：蒙恬科技（台灣老牌 OCR 廠商，1990 年代起）
- **產品線**：行動版（名片王 Mobile / Lite）+ 雲端版 + 桌面版
- **使用者基數**：累計數百萬

### 收費結構

| 方案 | 價格 | 限制 / 涵蓋 |
|------|------|------------|
| **Lite（免費）** | NT$0 | **70 筆資料**（50 辨識 + 20 匯入）；匯出僅姓名、電話、肖像 3 項 |
| **正式版（買斷）** | **NT$990 一次性** | 無限張數；完整匯出；個人化群組；雲端備份 |
| **雲端版（訂閱）** | **US$5/月 ≈ NT$160/月**（年繳） | 跨平台（iOS/Android/Win/Mac/Web）；訂閱到期 180 天後刪資料 |

### 策略觀察

1. **三軌並行**：免費鉤子（Lite）+ 一次性買斷（NT$990）+ 雲端訂閱（NT$160/月）
2. **70 筆免費**很聰明：個人輕度使用夠用，業務用戶必跨門檻
3. **匯出限制 = 軟性鎖定**：免費版資料拿不太出來，逼用戶要嘛留下，要嘛升級
4. **一次性買斷 NT$990 是台灣 sweet spot**：30 年來的價格錨點，使用者熟悉、不抗拒
5. **訂閱版主打的是「跨裝置同步」**而不是進階功能 — 因為核心功能買斷已經給了

### 對人脈手札的啟示

- ✅ NT$990 買斷是台灣本地市場最熟悉的價格 — 您原訂的 Lifetime NT$999 完全合理
- ✅ 「免費筆數限制」（您現規劃 20 人脈）思路正確，可以參考名片王再寬鬆一點到 30-50 人脈
- ✅ 雲端同步當「訂閱專屬功能」是清楚的差異化路徑（買斷 ≠ 訂閱）

## 二、先知命局（香港 SeerQ）— 收費機制

### 基本資料
- **公司**：香港 SeerQ Technology
- **定位**：AI + 數位平台 + 大師人脈媒合（全球數百位命理師）
- **覆蓋面**：八字、紫微、塔羅、占星、六爻、手面相、心理...

### 收費結構

#### 訂閱方案（會員）

| 方案 | 價格（美元） | 換算 NT$ | 備註 |
|------|-------------|---------|------|
| **週訂閱** | 首週 $2.99 → 後 $9.99/週 | 首週 NT$95 → 後 NT$320/週 | 釣魚式試水溫 |
| **月訂閱** | $19.99/月 | **~NT$640/月** | 高價位 |
| **年訂閱** | $39.9/年 | **~NT$1,280/年** | 比月訂閱便宜 83% |

#### 模組化訂閱（1-8 工具任選）

| 工具數 | 價格 | 換算 NT$ |
|--------|------|---------|
| 1 個工具 | US$3.9/月 | ~NT$125/月 |
| ⋮ | ⋮ | ⋮ |
| 8 個工具 | US$19.9/月 | ~NT$640/月 |

#### 加值消費（per-event）

- 大師諮詢（人工命理師服務）
- 大德靈符（電子靈符）
- 祈福台（單次儀式）

→ 都是**單筆收費**，價格依大師等級浮動

### 會員權益（訂閱才有）
- 每日 3 次「數位人」AI 測算
- 每日 1 次八字排盤報告
- 每週 3 份測算報告
- 每週 5 份大德靈符
- 每日 3 次祈願上香

### 免費功能（不訂閱也能用）
- 每日運勢（每日能量指數、幸運物、年月日運勢）
- 八字排盤（含十年大運、五行流轉）
- 黃曆擇日

### 策略觀察

1. **免費功能非常豐富**：八字排盤、每日運勢這些「基礎命理」都免費 — 因為這是流量來源
2. **付費賣的是「進階解讀 + 大師諮詢 + 靈符儀式」**：免費引流、付費深度
3. **模組化定價**是亮點：使用者可選 1-8 個工具，從 NT$125 起跳，門檻極低
4. **per-event 加值消費**：除了訂閱還能單筆買服務 — 變現密度高
5. **週訂閱當試水溫**：NT$95 首週讓使用者試用後再轉月年
6. **平台 + 媒合**：除了賣自己 AI，還抽大師諮詢佣金 — 雙引擎

### 對人脈手札的啟示

- ✅ **模組化收費**很值得學：把「深入識人」「雲端同步」「無限人脈」拆成獨立模組，使用者買最需要的那個
- ✅ **per-event 收費**是個人開發者最好的變現方式：不用建訂閱系統就能收錢
- ⚠️ **月訂閱 NT$640 是台灣使用者上限**：先知命局是大廠（有大師、客服、伺服器），個人開發者不能定那麼高
- ⚠️ **平台抽佣**不適合您（您沒有大師人脈，也不該往這方向走）

## 四、修正後的收費結構建議

### 為什麼要修正

原本規劃：
- Free（20 人脈）
- Pro NT$149/月（無上限 + 深入識人不限 + 雲端同步）
- Lifetime NT$999 限量 50 名

**問題與機會**：
1. Pro 全包 NT$149 對「只需要其中一個功能」的人是門檻
2. 沒有 per-event 選項 — 失去「我只想試一次」的衝動消費
3. Lifetime 比名片王貴 NT$9（無感差異）— 不如做 NT$890 製造「對比感」

### 修正版三層收費

#### Tier 0：Free（永久免費，是您的鉤子）

| 功能 | 包含 |
|------|------|
| 人脈管理 | **30 人脈**（從原本 20 提升，跟名片王 70 形成對比） |
| 命理分析 | ✅ 八字、紫微、星座、數字易經（全部免費！） |
| 合作分析 | ✅ 基本合作分析 |
| 雙向視角 | ✅ |
| 匯出 | ✅ JSON 完整匯出（區別自己「不鎖匯出」的隱私訴求） |
| 深入識人（AI） | ❌（要付費） |
| 雲端同步 | ❌ |

**為什麼這樣設計**：
- 命理分析完全免費 — 學先知命局把「基礎命理」免費引流，建立口碑
- 30 人脈足夠 1-2 個月試用，業務用戶會升級
- 匯出不鎖 — 因為您的差異化是「隱私不綁人」，匯出不通是反差異化

#### Tier 1：Lifetime 買斷 NT$890（推薦主力）

| 加贈 |
|------|
| ✅ 無限人脈 |
| ✅ 深入識人 AI（無限次） |
| ✅ 進階主題、深色模式、無廣告（如果之後有的話） |
| ✅ 終身免費所有未來功能（除了雲端同步） |
| ✅ 優先客服 |
| ❌ 雲端同步（要另加月訂閱） |

**為什麼是 NT$890 而不是 NT$999**：
- 跟名片王 NT$990 形成「比它便宜 NT$100」的對比錨點
- 心理上 NT$8XX 是「便宜一階」的價位段
- 賺得不多但**轉換率會高很多**（個人開發者要先把帳號做大）

#### Tier 2：Cloud Sync NT$99/月（追加訂閱）

| 包含 |
|------|
| ✅ 跨裝置雲端同步（手機 + 電腦） |
| ✅ 多裝置自動備份 |
| ✅ 包含 Lifetime 所有功能（不需另買 Lifetime） |

**為什麼這樣分**：
- 雲端同步是**真實有 server 成本**的功能 — 月訂閱合理
- 不需要雲端同步的人可以只買 Lifetime，月固定支出 = 0
- 想跨裝置的人可以單獨買 Cloud Sync = NT$99/月 = 一年 NT$1,188 比 Lifetime 還貴
- 兩種價格邏輯都自洽

#### Tier 3：Per-Event 深入識人 NT$30/次（衝動消費）

- 沒買 Lifetime 也可單次買「深入識人」分析
- NT$30/次 = 三杯零食的價格，極低門檻
- **10 次包**：NT$200（單價 NT$20）
- **50 次包**：NT$888（單價 NT$17.8）

**為什麼有這個**：
- 學先知命局的 per-event 模式
- 解決「我不確定要不要買 Lifetime，但想試一次」的猶豫
- 50 次包 NT$888 ≈ Lifetime 價格 — 自然引導消費者「直接買 Lifetime 划算」

## 五、修正後的損益平衡

### 原計算（單一訂閱）
107 付費 × NT$149/月 = **NT$15,943/月 MRR**

### 修正後預估（混合模式，第 8 月情境）

假設第 8 個月累積：
- Lifetime 買斷：**150 名**（年累積，貢獻 NT$890 × 150 = NT$133,500 一次性）
- Cloud Sync 訂閱：**40 人**（NT$99/月 × 40 = **NT$3,960/月 MRR**）
- Per-event 深入識人：**月銷 100 次**（平均單價 NT$25 × 100 = **NT$2,500/月**）
- 贊助：~NT$1,000/月

**月度比較**：

| 模式 | 月經常性收入 | 一次性收入（攤平年化） | 月總收入 |
|------|-------------|---------------------|---------|
| 原訂閱模式 | NT$15,943 | NT$0 | NT$15,943 |
| 修正混合模式 | NT$7,460 | NT$11,125（NT$133,500 ÷ 12） | **NT$18,585** |

**達標時程修正**：

- 原規劃：第 8 月達 107 付費 → 平衡
- 修正後：第 5-6 月若 Lifetime 賣 80 名 + 40 個 Cloud Sync → 已達平衡
- **預估比原方案提早 2-3 個月達標**（因為 Lifetime 一次性現金流大）

### 但要注意的風險

- Lifetime 的「年化攤平」是會計概念，**實際現金流是前置**的：
  - 前 6 月：賣 80 名 Lifetime = NT$71,200 一次入帳 ✓ 很爽
  - 第 13 月後：Lifetime 銷售減速，得靠 Cloud Sync MRR 撐
- **Cloud Sync 必須真的有用**：如果使用者不需要跨裝置，這個 SKU 賣不掉
- **per-event 需要付費系統**：跟訂閱用同一套金流（Lemon Squeezy / 綠界）

## 六、修正後的執行藍圖（覆蓋原 `變現執行藍圖_12個月.md`）

### M0-M3 完全不變

朋友圈試用 + 軟啟動。這階段不收費，重點是驗證留存與口碑。

### M4-M5 修正（增長啟動）

**原版**：上 Lifetime NT$999 × 50 名

**修正版**：
- 上 Lifetime **NT$890**（不設限量，賣多少就多少）
- 同時上 **per-event 深入識人 NT$30/次**（讓沒準備買斷的人也能體驗）
- 推薦獎勵：推薦一人購買 Lifetime → 推薦人得 NT$200 信用點
- **第 4 月底目標**：50 Lifetime + 50 per-event 嘗試者

### M6-M8 修正（上付費）

**原版**：上 Pro NT$149/月

**修正版**：
- **不再做 Pro 全包訂閱**，改上 **Cloud Sync NT$99/月**（單獨功能）
- Lifetime 持續銷售（無限量）
- Per-event 持續
- 第 8 月目標：150 Lifetime + 40 Cloud Sync 訂閱者 + 月 100 次 per-event

### M9-M12 修正（規模化）

**新增 SKU 探索**：
- M9：年度深度命盤分析 NT$299 一次性
- M10：B2B 5 人團隊版 NT$499/月（5 人共享）
- M11：雙人合盤報告 NT$50 一次性
- M12：Lifetime 漲價到 NT$1,099（因為已經累積 200+ 早鳥）

## 八、結論：差異化定位的最終一句

> **人脈手札 = 名片王的「結構」+ 先知命局的「深度」+ 個人開發者的「便宜」**

- 比名片王：**多了命理 + 深入識人**（差異化）
- 比先知命局：**多了人脈管理 + 隱私本機儲存**（差異化）
- 比兩者：**Lifetime NT$890 比名片王便宜 NT$100，月訂閱 NT$99 比先知命局便宜 NT$540**（價格優勢）

這三個方向疊在一起，就是您可以打的故事 — **「便宜很多，但功能更獨特」**。
